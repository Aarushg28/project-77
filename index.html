Project About Console


Visual Studio Code is an integrated development environment made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.


The Web Console: Logs information associated with a web page warning and informational messages explicitly logged by JavaScript code running in the page context. You can open the console screen by right clicking on the screen and selecting inspect option. We can print text in console using a java script syntax (console.log).
